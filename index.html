Loads a configuration preset.
"Last session" restores the options used in the previous session. Saves the current configuration options to the "Custom" preset.
FFT size: Number of samples used for the FFT. Larger values will provide greater detail on lower frequencies, but will require more CPU power.
Range: - Hz
Smoothing: Lower values make the analyzer respond faster to changes. Increase it if the animation looks too "jumpy".
Gradient:
  • Log Use a logarithmic scale to enhance visualization of low frequencies
  • Scale Show / hide the frequency scale
  • Sens Increase analyzer sensitivity to improve visualization of songs too quiet
  • Peak Hold volume peaks on screen for a short time
